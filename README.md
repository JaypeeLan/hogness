# Hogens Technologies Ltd — Corporate Website

![Hogens Technologies Ltd](/public/logo.png)

Hogens Technologies Ltd is a technology company dedicated to making learning and technology simple, practical, and accessible. This platform provides hands-on training programs, digital education solutions, and technology consulting to help people understand and use technology with confidence.

## 🚀 Vision & Mission

- **Vision:** To contribute to a future where technology education is accessible, practical, and empowering for everyone.
- **Mission:** To empower individuals, institutions, and organizations with practical technology skills and digital knowledge that support growth, innovation, and informed decision-making.

## 🛠 Tech Stack

- **Core:** [Next.js 15](https://nextjs.org/) (App Router)
- **UI Logic:** [React 19](https://react.dev/)
- **Styling:** [SCSS Modules](https://sass-lang.com/) with a custom design token system.
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Interactivity:** Next.js Client Components for dynamic sections.
- **Progress:** [nextjs-toploader](https://www.npmjs.com/package/nextjs-toploader) for smooth page transitions.

## ✨ Key Features

- **Dynamic Course Engine:** Pages for Data Analysis, AI/ML, Data Science, Python, Digital Literacy, and Coding for Kids.
- **Premium Design:** Implements modern web design patterns including hero overlaps, glassmorphism-influenced cards, and smooth animations.
- **Responsive Layout:** Truly mobile-first design with a functional hamburger menu and adapted container widths.
- **Optimized Performance:** Static Site Generation (SSG) for fast page loads and SEO benefits.

## 📂 Project Structure

```text
src/
├── app/            # Next.js App Router pages & SCSS modules
├── components/     # Reusable UI parts & shared sections
├── styles/         # Global SCSS, tokens, and mixins
├── lib/            # Shared data and utility functions
└── types/          # TypeScript definitions
```

## 🛠 Getting Started

First, install the dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📦 Building for Production

To create an optimized production build:

```bash
npm run build
npm start
```
